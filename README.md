# getablog.org

A website built to encourage others to start a blog.

Currently very minimal, hosting a little note of encouragement and a list of useful resources. 

In the future, it may host more content or tooling that's helpful for blog writers.

## Contributing

Contributions are more than welcome for resources that can help one with a blog.

If you have any more ideas for this website can do, let me know in an issue and we can talk about it.
